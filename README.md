# AnimeLens

An app to where you can get all the anime details. **Thanks to [AniList](https://anilist.co) for providing graphql api for this project.**

## Commands

1. Run development server
   ```bash
   npm run dev
   ```
2. Build project
   ```bash
   npm run build
   ```
3. Run linter
   ```bash
   npm run lint
   ```
4. Fix lint issues
   ```bash
   npm run lint-fix
   ```
5. Check typescript errors
   ```bash
   npm run ts-check
   ```
